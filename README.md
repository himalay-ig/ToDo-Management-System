# ToDo-Management-System
A CRUD operations–based ToDo application developed as a collaborative project by me and Aryak, focused on efficient task management.
# How to run:
Step 1: git clone https://github.com/himalay-ig/ToDo-Management-System.git 

Step 2: cd ToDo-Management-System

Step 3: docker-compose up

Then open:
http://localhost:3001
